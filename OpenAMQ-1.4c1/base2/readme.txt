This is the iMatix "Base2" technology toolkit, and consists of the
following products:

- XNF, a code generator that builds code generators.
- iCL, a class system for object-oriented literal programming.
- iPR, a re-tooling of the old iMatix SFL portability library.
- OPF, a code generator for configuration data sets.
- SMT, a framework for building high-performance multithreaded servers.
- ASL, a framework for building protocol servers and clients.
- Xitami/5, an embeddable web server.

Please see the individual product's readme.txt files for more
information, including licensing and copying conditions.
