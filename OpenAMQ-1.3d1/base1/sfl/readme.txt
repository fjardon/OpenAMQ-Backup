SFL is the iMatix Standard Function Library.  This is a C library
containing functions for portability between Unix, OS/2, Windows, and
OpenVMS.  SFL also contains utility functions for commonly features such
as memory management, list handling, hashing, and XML processing.  SFL
is fully portable to most modern operating systems.

SFL forms part of the iMatix "Base" technology toolkit, and is provided
under the GNU General Public License (GPL) for free software developers,
and the iMatix General Terms of Business for commercial developers.

Please note that this is a legacy version of SFL provided for backward
compatibility only, and is no longer being maintained.  For more
information please see http://www.imatix.com/.
