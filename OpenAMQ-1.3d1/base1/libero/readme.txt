Libero is a finite-state machine code generator.  It will generate a
control framework for a high-level program logic definition called a
'dialog'.  A Libero dialog is a simple and effective way of describing
the logic of all kinds of programs.  Libero generates code in many
languages and you can easily add your own.  It has been in active use
since 1991.
